Switching to or from Live

The Connect simulator in your Developer Playground is for test transactions. The Connect live gateway is for real-money transactions.

Switching from test transactions to live transactions is a simple as changing a single URL in your code. And vice versa.

{% include docs/components/text-block-divider.html %}

Test URL

The URL for sending test transactions to the Connect simulator should look like the following:

{% include docs/components/code-block.html content="https://api.testingpays.com/API_KEY/authipay/v1/connect" language="http"%}

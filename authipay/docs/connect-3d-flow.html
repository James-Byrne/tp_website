Testing 3D Secure Flows

The Connect simulator offers you the ability to try out 3D secure flows. Use the simulator to test and understand the customer experience when 3D secure is enabled for a Connect transaction.

To initiate the 3D secure flow, you need to include the authenticateTransaction parameter with the value ‘true’ in your requests.

If this is received, the simulator will redirect you to a simulated bank verification screen (also known as an ACS screen). On this screen, you can then select continue to successfully complete the 3D transaction, or select cancel to end the transaction.

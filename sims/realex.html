Realex

You can get going with our Realex sims within minutes of signing up, no need to wait for API Keys and access codes. Follow our example apps and code to start off your integration using Ruby, Java, PHP, etc.

Our realex sims include:

  • Auth - for one-off transactions
  • 3DS-verifyenrolled - 3D secure (verified by Visa) flow starting point
  • 3DS-verifysig - 3D secure (verified by Visa) flow callback after the bank page

With the sims above you can get your code off the ground, and prepare for hard-to-cover scenarios like a complete 3D secure flow and card-currency mismatch!

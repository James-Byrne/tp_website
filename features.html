Platform Features

Your toolkit for developing and testing payments

Payment Gateway Simulators

The Testing Pays platform provides a set of tools for tech teams to develop and operate top class payment channels on Online Merchant sites. Troubleshoot complex card payment issues, improve your customers' checkout experience and maintain top quality, always-on online payments software code.

Reporting Dashboard

All a team's activity is captured and displayed in real-time team dashboards. Whether a team of one or many, you can keep track of your test coverage across your all important payments channels.

On-demand API Responses

Testing Pays allows users to generate any API response for some of the world's leading payment and fraud providers. Our API mapping tables map each API response code to transaction values - make a test payment with a specific cent or pence amount and get back the response code you desire for the API your testing.

Catch any bottle necks

You need to put your payments software under load pressure to characterise it's performance and catch any bottle necks.

Dedicated Environment

Our performance harness provides you with a dedicated environment in which you can run big loads of test transactions through your e-commerce software.

Successful Transactions

Increase the amount of successful transactions by making sure you have less checkout errors, better failure recovery and minimised software down-time.

Response Parameter Templates

Configure and lock-in the exact response parameters that reflect the contracturals between your company and the payment provider. Use the template in all regression testing to ensure that no bad business configurations make it into the Production environment, where they could lead to massive mischarging of processor fees.

Custom Responses

Configure your own custom responses. Copy and paste unexpected 3rd party responses from live Production environments into TestingPays and hey presto, you're able to recreate the scenario endlessly.

Transaction Logging

Well-structured logging is essential to tracing and troubleshooting payments. Our offline and 'live tail' logging were built to provide software teams with an easy way to trace the their test transactions.

Full Documentation

Our documentation has been uniquely crafted to make testing as easy as possible. We supply you with code snippets, useful response codes and simple integration instructions.
